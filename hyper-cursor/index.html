FAQ

Why isn't Hyper Cursor working on some pages?
+

Chrome extensions cannot run on certain protected pages for security reasons:

  • chrome:// pages (Settings, Extensions, etc.)
  • chrome-extension:// pages
  • Chrome Web Store pages
  • Some browser internal pages

This is a Chrome security restriction and applies to all extensions. Hyper Cursor will work perfectly on all regular websites!

How do I uninstall Hyper Cursor?
+

To remove Hyper Cursor from Chrome:
